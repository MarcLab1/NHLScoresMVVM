# NHLScoresMVVM

A simple app that displays scores of the 2021-2022 NHL regular season

## Concepts 

  * Kotlin
  * Jetpack Compose
  * Composables
  * Coroutines 
  * Retrofit2
  * GSON 
  * Hilt
  * Lifecycle
  * JodaTime
  * Coil
  * Compose Navigation
  * MVVM
